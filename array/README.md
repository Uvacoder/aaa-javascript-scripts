# array
array are used to store multiple values in single variable
```javascript
var teamName = ["james", "erina", "erick", "blawzkovic"];
```
```javascript
var teamName = [
  "james",
  "erina",
  "erick",
  "blawzkovic"
];
console.log(teamName[0]); //output "james"
```
